# Correção — Serviços Contratados no Prontuário

Arquivo alterado:

- public/pages/alunos/prontuario.js

Objetivo:

Vincular a aba Serviços Contratados aos dados que o aluno já possui quando é matriculado.

Agora a aba monta automaticamente:

- Plano contratado
- Valor mensal
- Situação da matrícula
- Próximo vencimento
- Renovação automática/manual
- Modalidades
- Serviços extras, quando existirem no cadastro/matrícula/financeiro
- Professor responsável
- Turma
- Avaliação física
- Treino ativo
- Benefícios básicos
- Resumo financeiro

Também foi preenchida a aba Contrato Comercial com o resumo base do contrato do aluno.

Observação:

Esta etapa usa os dados já existentes no prontuário. Não cria novos módulos de serviços extras ainda.
