# Fusion ERP 2.7.5 — ZIP B3 Menu Global Accordion

## Objetivo
Evoluir o menu global para navegação de ERP com grupos recolhíveis, mantendo a estrutura aprovada no ZIP B2.

## Arquivos alterados
- `public/assets/js/fusion-layout.js`
- `public/assets/css/fusion-menu-global.css`

## Ajustes incluídos
- Grupos do menu recolhíveis por clique.
- Estado dos grupos salvos automaticamente no navegador.
- Grupo da página atual abre automaticamente.
- Remoção do rodapé “Layout global”.
- Destaque visual mais claro para o módulo ativo.
- Animação suave de abertura e fechamento.
- Comportamento mobile preservado como gaveta lateral.

## Aplicação
Extrair este ZIP na raiz do projeto e substituir os arquivos.
Depois reiniciar o servidor:

```bash
CTRL + C
npm start
```

No navegador, limpar cache:

```text
CTRL + F5
```
